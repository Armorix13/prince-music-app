## Usage Instructions

1. **For Free Courses**: Use `courseType: 2` and omit the `price` field
2. **For Paid Courses**: Use `courseType: 1` and include the `price` field
3. **Required Fields**: All fields shown are required except `price` (only for paid courses)
4. **Testing**: Start with minimal examples for quick testing, then use comprehensive examples for full features

## API Endpoint
**POST** `/api/v1/courses`

Use any of these examples in your Postman requests to test the course creation functionality!
